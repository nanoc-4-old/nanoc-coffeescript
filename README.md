# nanoc-coffeescript

This provides a [CoffeeScript](http://coffeescript.org/) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-coffeescript`

## Usage

```ruby
filter :coffeescript
```

This filter takes no options.
